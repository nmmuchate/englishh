Key architecture decisions:
- QStepper with `v-model="step"` (string values: 'welcome', 'assessment', 'result')
- `flat` prop (no QCard shadow)
- `:header-nav="false"` prop (disables clickable step tabs)
- `:deep(.q-stepper__header) { display: none }` in scoped style (hides the tab row visually)
- Navigation via `step.value = 'assessment'` (direct assignment, clean and simple)
- Radio cards: QItem tag="label" + QRadio (NOT QOptionGroup)
- All icons use sym_o_ prefix (material-symbols-outlined)
