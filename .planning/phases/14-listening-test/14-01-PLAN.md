  <action>
Create `src/pages/ListeningPlayer.vue` — a self-contained TTS playback component.

**Props:**
- `text` (String, required) — the prompt text to speak
- `maxReplays` (Number, default: 1) — number of replays allowed after initial play
- `disabled` (Boolean, default: false) — external disable (e.g., while loading)

**Emits:**
- `played` — fired when play() starts (first or replay)
- `ended` — fired when utterance finishes

**Reactive state:**
- `isPlaying` (ref, false) — true while TTS is speaking
- `playCount` (ref, 0) — increments on each utterance end
- `ttsAvailable` (ref, true) — set in onMounted via `'speechSynthesis' in window`

**Computed:**
- `canPlay` — `!isPlaying && playCount === 0 && ttsAvailable && !props.disabled`
- `canReplay` — `!isPlaying && playCount > 0 && playCount <= props.maxReplays && ttsAvailable && !props.disabled`
- `playLabel` — `isPlaying ? 'Playing...' : playCount === 0 ? 'Play Audio' : 'Replay'`

**Methods:**
- `play()`:
  1. Guard: `if (isPlaying.value || props.disabled) return`
  2. Guard: `if (playCount.value > props.maxReplays) return`
  3. `window.speechSynthesis.cancel()` — clear any in-progress utterance (Pitfall 1 prevention)
  4. Create `new SpeechSynthesisUtterance(props.text)`
  5. Set `utterance.lang = 'en-US'`
  6. Set `utterance.rate = 0.9` — slightly slower for comprehension
  7. `utterance.onstart = () => { isPlaying.value = true }`
  8. `utterance.onend = () => { isPlaying.value = false; playCount.value++; emit('ended') }`
  9. `utterance.onerror = () => { isPlaying.value = false }` — Pitfall 1 safety
  10. `window.speechSynthesis.speak(utterance)`
  11. `emit('played')`

**Lifecycle:**
- `onMounted`: `ttsAvailable.value = 'speechSynthesis' in window`
- `onUnmounted`: `window.speechSynthesis.cancel()` — Pitfall 3 prevention

**Watch:**
- `watch(() => props.text, () => { playCount.value = 0; window.speechSynthesis.cancel() })` — reset when task changes

**Template:**
```html
<template>
  <div class="listening-player column items-center q-gutter-sm">
    <!-- TTS unavailable fallback -->
    <template v-if="!ttsAvailable">
      <q-banner class="bg-orange-1 text-orange-9 full-width q-mb-sm" rounded>
        Audio playback is not available in this browser.
      </q-banner>
    </template>

    <!-- Play / Replay button -->
    <q-btn
      v-if="ttsAvailable"
      :icon="isPlaying ? 'volume_up' : 'play_circle'"
      :label="playLabel"
      :color="isPlaying ? 'grey' : 'primary'"
      :disable="isPlaying || (!canPlay && !canReplay)"
      unelevated
      rounded
      no-caps
      size="lg"
      class="listening-play-btn"
      @click="play"
    />

    <!-- Replay count hint -->
    <span v-if="playCount > 0 && ttsAvailable" class="text-caption text-grey-5">
      {{ playCount > maxReplays ? 'No replays remaining' : `${maxReplays - playCount + 1} play(s) remaining` }}
    </span>
  </div>
</template>
```

**Scoped CSS:**
```css
.listening-player {

}

  min-width: 200px;
}
```
