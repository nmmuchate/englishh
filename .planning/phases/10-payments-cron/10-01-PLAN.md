Implementation steps:

1. Add a helper `verifyMozPaymentsSignature(req, secret)`:
   - Reads `req.headers['x-mozpayments-signature'] || ''`
   - Uses `req.rawBody || Buffer.from(JSON.stringify(req.body))` as the body to hash (rawBody is a Buffer in deployed functions; fallback allows emulator testing — NOT production-safe, document with a comment)
   - Computes `crypto.createHmac('sha256', secret).update(rawBody).digest('hex')`
   - Compares using `crypto.timingSafeEqual(Buffer.from(signature, 'utf8'), Buffer.from(computed, 'utf8'))` — always check lengths first to avoid timingSafeEqual crash
   - Returns boolean

2. Implement `exports.handlePaymentWebhook = onRequest({ secrets: [MOZPAYMENTS_API_KEY] }, async (req, res) => {...})`:

   - Log incoming event: logger.info('handlePaymentWebhook', { event, subscriptionId })
   - If event === 'payment.success':

     f. Update users doc: `admin.firestore().doc(\`users/${userId}\`).update({ subscriptionStatus: 'active', subscriptionExpiresAt: expiresAt, updatedAt: now })`
     g. Run both updates in parallel with Promise.all([...])
