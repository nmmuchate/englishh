function animateValue(refTarget, target, duration = 800) {
  const start = Date.now()
  const interval = setInterval(() => {
    const elapsed = Date.now() - start
    const progress = Math.min(elapsed / duration, 1)
    const eased = 1 - Math.pow(1 - progress, 3) // ease-out cubic
    refTarget.value = Math.round(eased * target)
    if (progress >= 1) clearInterval(interval)
  }, 16)
}

onMounted(() => {
  setTimeout(() => {
    animateValue(pronPct, 85)
    animateValue(gramPct, 70)
    animateValue(vocabPct, 90)
  }, 300)
})

Key implementation notes:
- `tab` is a local `ref('overview')` bound to both `QTabs` v-model and `QTabPanels` v-model — this is the standard Quasar tab switching pattern (FEED-02)
- `QTabs` uses `align="justify"` to spread tabs full width; `no-caps` keeps the label text as written
- `QTabPanels` must have `animated` for smooth transitions; set `class="feedback-tab-panels"` with `background: transparent` to inherit page bg
- Circular progress uses reactive refs `pronPct`, `gramPct`, `vocabPct` (all start at 0) bound via `:style="{ '--pct': pronPct + '%', '--clr': '#...' }"`. In `onMounted`, `animateValue()` uses a `setInterval` at 16ms with an ease-out cubic curve to increment each ref from 0 to its target (85/70/90) over 800ms — this drives the conic-gradient fill animation cross-browser (FEED-01).
- `session.overallScore` is read from the session store and displayed as a large bold primary-colored number directly below "Session Complete!" in the hero card — this closes the loop with `endSession(82)` called in SessionPage.vue
- `goToDashboard()` calls `router.push({ name: 'dashboard' })` — used by both the X close button (FEED-04) and the footer button
- `useSessionStore()` is legitimately used: `session.overallScore` renders in the hero card
- Do NOT use QLinearProgress for tab indicator — QTabs handles that internally
- Do NOT add any Tailwind classes — all styling is scoped CSS
- The `sym_o_book_4` icon may not be in all Quasar icon sets — if it causes a warning substitute `sym_o_menu_book`
